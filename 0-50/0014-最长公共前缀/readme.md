## 题目 
- [题目](https://leetcode-cn.com/problems/longest-common-prefix/)

## 描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

## 示例 
**示例1**:
```text
输入: ["flower","flow","flight"]
输出: "fl"
```

**示例2**:
```text
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

**说明**:
所有输入只包含小写字母 a-z 。
