# 二进制中1的个数

## 题目描述
输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

## 左移 右移
### 左移
左移运算符 m << n 表示把 m 左移 n 位。左移 n 位的时候，
最左边的 n 位将被丢弃，右边补0；相当于乘 2
```text
00001010 << 2 = 00101000
10001010 << 2 = 00101000
```

### 右移动
右移运算符 m >> n 表示把 m 右移动 n 位。右移 n 位的时候，分两种场景；<br>
1. 如果数字是无符号数值，则用0填补最左边的 n 位；相当于除 2
2. 如果数字是负数，则右移之后，在最左边补充 n 个 1;
```text
00001010 >> 2 = 00000010
10001010 >> 2 = 11100010

```
