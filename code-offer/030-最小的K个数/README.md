## 题目描述
```text
输入n个整数，找出其中最小的K个数。
```

## 样例
### 输入
```text
4,5,1,6,2,7,3,8

4
```

### 输出
```text
1,2,3,4
```

## 思路
### 方法一：排序
要求一个序列中最小的K个数，按照惯有的思维方式，很简单，先对这个序列从小到大排序，然后输出前面的最小的K个数即可

### 方法二：选择或者交换排序
再进一步想想，题目并没有要求要查找的k个数，甚至是后面的n-k个数是有序的，既然这样，咱们又何必对所有的n个数都进行排序呢？
