## 题目描述
```text
地上有个 m 行 n 列的方格。一个机器人从坐标(0,0)的格子开始移动，它每一次可以向左、右、上、下移动一格，但不能进入行坐标和列坐标的数位之和大于 k 的格子。
```

## 示例
```text
当 k 为 18 时，机器人能够进入方格(35,37)，因为 3+5+3+7=18 
但它不能进入方格(35,38)，因为 3+5+3+8=19 请问该机器人能够达到多少格子？
```
