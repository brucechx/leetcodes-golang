# [题目描述](https://leetcode-cn.com/problems/bitwise-and-of-numbers-range/)
给定范围 [m, n]，其中 0 <= m <= n <= 2147483647，返回此范围内所有数字的按位与（包含 m, n 两端点）。

- 示例 1: 
```text
输入: [5,7]
输出: 4
```

- 示例 2:
```text
输入: [0,1]
输出: 0
```
